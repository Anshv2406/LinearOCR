# LinearOCR: Handwritten Text Recognition

## 📖 Description
A deep learning-based solution designed to perform Handwritten Text Recognition (HTR). This project processes handwritten images and converts them into machine-encoded text using a combination of Convolutional Neural Networks (CNN) and Recurrent Neural Networks (LSTM).

## 🚀 Key Features
* **Model Architecture:** Utilized a CNN-LSTM hybrid architecture for feature extraction and sequence modeling.
* **Training:** Successfully trained over 60+ epochs to optimize character recognition accuracy.
* **Preprocessing:** Implemented image processing techniques to enhance handwritten character visibility.

## 🛠 Tech Stack
* **Language:** Python
* **Deep Learning Frameworks:** TensorFlow, Keras
* **Libraries:** NumPy, OpenCV, Matplotlib, Pandas
* **Environment:** Google Colab

## 📂 Project Structure
* `DLPRO (1).ipynb`: The primary notebook containing data preprocessing, model architecture definition, training loops, and evaluation metrics.

## 📈 Results
* Average CER: 11.08%
* Average WER: 35.12%

## 💻 How to Run
1. Click the file `DLPRO (1).ipynb` in this repository.
2. Click the **"Open in Colab"** badge or button (if present) or simply copy the code to your local environment.
3. Ensure the dataset is loaded in the path specified in the notebook.
4. Run the cells sequentially to observe the training/prediction process.
